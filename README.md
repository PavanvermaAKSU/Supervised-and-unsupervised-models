# Supervised and Unsupervised Machine Learning Models

This project contains implementations of basic Machine Learning algorithms in Python for both **Supervised** and **Unsupervised Learning**.

The programs are written using **NumPy** and **Scikit-learn** libraries and demonstrate how different ML algorithms work with runtime input.

---

## 📌 Project Structure

```
Supervised-and-unsupervised-models
│
├── supervised_ML
│   ├── linearReg.py
│   ├── logisticReg.py
│   ├── polynomial.py
│   └── KNN.py
│
└── unsupervised_ML
    └── Kmeans.py
```

---

## 📊 Supervised Learning Algorithms

Supervised learning uses **labeled data** to train the model.

Implemented algorithms:

* Linear Regression
* Logistic Regression
* Polynomial Regression
* K-Nearest Neighbors (KNN)

These algorithms are used for **prediction and classification problems**.

---

## 📊 Unsupervised Learning Algorithms

Unsupervised learning works with **unlabeled data** and finds patterns or clusters.

Implemented algorithm:

* K-Means Clustering

It groups similar data points into clusters.

## ▶️ How to Run

1. Install required libraries

```
pip install numpy scikit-learn
```

2. Run any program

Example:

```
python supervised_ML/linearReg.py
```

or

```
python unsupervised_ML/Kmeans.py
```
